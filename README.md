# 🔐 Password Generator

A modern and secure Password Generator built using HTML, CSS, and JavaScript.

## ✨ Features

* Generate strong and secure passwords
* Custom password length selection
* Include numbers option
* Include special symbols option
* One-click password copy feature
* Interactive and animated user interface
* Glowing visual effects
* Dynamic quote animations
* Responsive design for desktop and mobile devices
